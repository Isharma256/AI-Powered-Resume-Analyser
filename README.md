 AI Powered Resume Analyzer

## Description
This project analyzes resumes using AI techniques and provides feedback to improve content quality and job matching.

## Features
- Resume analysis
- Skill extraction
- Suggestions for improvement
- User-friendly interface

## Technologies Used
- Python (Flask)
- HTML, CSS
